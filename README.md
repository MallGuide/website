# MallGuide Website

Static marketing site for MallGuide.

## Files

- `index.html` - landing page
- `privacy.html` - privacy policy
- `terms.html` - terms of use
- `css/styles.css` - custom styles and animations
- `js/script.js` - navigation, reveal effects, FAQ behavior
- `assets/logo.svg` - brand mark
- `assets/favicon.svg` - favicon

## Local preview

Open `index.html` in your browser, or serve this folder with any static server.
